# FizzBuzz
Given a number <i>n</i>, for each integer <i>i</i> in the range from <i>1</i> to <i>n</i> inclusive, print one values per line as follows:<br/><br/>
<ul>
<li>If <i>i</i> is a multiple of both <i>3</i> and <i>5</i>, print <i>FizzBuzz</i>.</li>
<li>If <i>i</i> is a multiple <i>3</i> (but not <i>5</i>), print <i>Fizz</i>.</li>
<li>If <i>i</i> is a multiple <i>5</i> (but not <i>3</i>), print <i>Buzz</i>.</li>
<li>If <i>i</i> is not a multiple of <i>3</i> or <i>5</i>, print the value of <i>i</i>.</li>
</ul>
<br/><br/>
<b>Constraints</b><br/>
<i>0 < n < 2 * 10<sup>5</sup>
